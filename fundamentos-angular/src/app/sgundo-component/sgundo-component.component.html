Interpolação de texto

Interpolação de texto, é forma como você consegue inseri informações da classe para dentro do template.
Primeiro você vai na classe do componente, neste caso o "segundo-component.componet.ts",
coloca a variável, contendo o que você deseja mostrar na página, veja a imagem:

No "segundo-component.component.html", você pode ultilizar qualquer tag, e colocar "chaves" duplas e o nome da variável, que você quer mostrar no template ou na página.

Como fica na página:

Nome: {{ nome }}

Cidade: {{ cidade }}

Idade: {{idade}}


Usar o Pipe "|" para fazer formatações no texto

Você coloca o texto dentro das chaves duplas e o pipe, como está a imagem abaixo:
Do lado do piper você passa a formatação que você quer que seja feita, neste caso está deixando todo texto com letras Maiúsculas.

Como fica na página:

Nome: {{ nome | uppercase }}
