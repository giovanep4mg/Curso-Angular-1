Roteamento entre págins, ultilizando o "angular-routes".
Vai ter que criar um novo projeto:

Crie um novo componente pelo cmd, usando esses comandos:ng new roteamento.
Quando pedir para você escolher se quer instalar o angular routing, você diz que não.
Escolha o CSS, vai demorar um pouquinho para instalar.
Após a instalação, você verifica se está mesmo na página "roteamento".
E coloca esse comando que vai instalar a biblioteca do angular-routes:
npm i @angular/router.
Cria um módulo para trabalhar com roteamento, com esse comando ng module app-routing app.
