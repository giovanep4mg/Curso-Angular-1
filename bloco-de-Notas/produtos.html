Produtos

Crie um novo componente, mas é um modulo = module, com o nome produtos.
Module, por que deixará mais rápido o carregamento da página.
Já acrescenta uma rota para ele, e atrele ele ao modulo app.
O comando é ng g module produtos --route produtos --module app.module Depois de criado, vá ate o app-routing.module.ts, e verifica se foi criado uma rota para produtos.

Para que ao abrir o site já abra essa página, vamos fazer isso:
No componente app-routing.module.ts, crie uma nova rota, como está na imagem abaixo

Agora vamos criar a página de produtos o seu conteúdo, no produtos.component.html, e adicione essas tags:
"section" classe = "product-list", essa seção é sobre os produtos
"div" classe = "product__list__card",
Crie um link "a" classe ="product-list__link,sem o href poid será um link diferente
"img", sem o alt,
"h2", com a classe ="product-item__name",
Coloque o nome
"p" class="product-list__price",
Adicione um valor, o preço e pode fechar a tag "p"
"p" class ="product-item-price__description",
Coloque o texto:"á vista",
Fecha a tag "p"
"button" classe="product-item_buy-button" type="button", adiciona o texto = "comprar"

Para simular um banco de dados, com os produtos, imagens, preços, tudo já está na pasta,
basta copiar e adicionar no projeto, o arquivo se chama produtos.ts,
copiar e colar dentro da pasta "produtos", assim você terá seu banco de dados.

Atenção: os arquivos estão na pasta "Aula-6-E-criando pagina de produtos".

Para colocar os estilos da página, é só fazer a mesma coisa que fez antes.
Copiando o arquivo produtos.component.css e substituindo o arquivo original do projeto.

As imagens dos produtos, também está nessa pasta e deve ser copiada e adiconada no projeto.
As imagens estão na pasta "Aula-6-E-criando.." , com o nome assets.
Copie essa pasta e subtituia pela original do projeto.
